# Bonsai Image Studio - 로컬 AI 이미지 생성기

## 1. 프로젝트 목적
이 프로젝트는 Hugging Face의 `prism-ml/bonsai-image-ternary-4B-gemlite-2bit` 모델을 로컬 환경(Windows PC)에 설치하고, 누구나 쉽고 빠르게 고품질의 이미지를 생성해 볼 수 있는 아름다운 웹 기반 인터페이스를 구축하는 것을 목적으로 합니다.

## 2. 왜 이 프로젝트가 필요한가?
- **로컬 실행 및 개인정보 보호:** 클라우드 서버에 프롬프트를 전송하지 않고 사용자의 PC에서 직접 이미지를 생성하여 개인정보 노출 우려가 없습니다.
- **초경량 최적화:** FLUX.2 Klein 4B 모델을 삼진수(2-bit) 양자화하여, 일반적인 소비자용 그래픽 카드(VRAM 2GB~6GB)를 탑재한 개인 PC에서도 빠른 속도로 고해상도 이미지 생성이 가능합니다.
- **교육용 도구:** AI 기초 교육과정에서 복잡한 코드나 설치 장벽 없이 직관적으로 이미지 생성 과정을 학습하고 실습해 볼 수 있습니다.

## 3. 주요 기능
- **프리미엄 다크 테마 UI:** 세련된 Glassmorphism 테마와 마이크로 애니메이션 피드백을 지원하는 프론트엔드.
- **다양한 이미지 비율 지원:** 1:1 정사각형, 16:9 가로형, 9:16 세로형 등 화면 비율의 원클릭 선택 기능.
- **세부 추론 제어:** 이미지의 화질을 결정하는 Steps(추론 단계)와 Prompt CFG Guidance 조절 기능.
- **시드 고정 및 랜덤화:** 일관된 이미지 변형을 생성할 수 있는 시드(Seed) 제어 기능.
- **안전 제어 및 최적화:** 저용량 VRAM 환경에서의 메모리 초과(OOM) 방지를 위한 Tiled VAE 기능 토글 지원.
- **성능 메타데이터 제공:** 이미지 생성에 걸린 시간(초 단위) 및 VRAM 피크 점유율을 실시간으로 화면에 렌더링.

## 4. 실행 방법 (단계별 상세 설명)
1. **PowerShell 스크립트 실행 권한 확인:**
   Windows PowerShell을 관리자 권한으로 열고 아래 명령어를 실행하여 스크립트 실행이 가능하게 설정합니다:
   ```powershell
   Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
   ```
2. **로컬 백엔드 서버 가동:**
   `Bonsai-Image-Demo` 폴더 내의 실행 스크립트를 통해 API 서버를 켭니다:
   ```powershell
   cd Bonsai-Image-Demo
   .\scripts\serve.ps1
   ```
   *서버가 성공적으로 구동되면 `http://localhost:8000` 포트에서 대기합니다.*
3. **웹 화면 접속:**
   - 프로젝트 폴더의 `image_generator.html` 파일을 더블 클릭하여 웹 브라우저에서 실행합니다.
   - 상단 우측의 서버 연결 상태 표시등이 초록색(`서버 연결 완료`)으로 변경되었는지 확인합니다.
4. **이미지 생성:**
   - 원하는 프롬프트를 한글 또는 영어로 작성한 뒤 `이미지 생성` 버튼을 누릅니다.

## 5. 환경 설정 방법
- **필수 사양:** Windows OS, NVIDIA 그래픽 카드 (MX450 이상 또는 RTX 계열 추천, VRAM 최소 2GB 이상).
- **의존성 설치:**
  `Bonsai-Image-Demo` 디렉토리 내의 `setup.ps1`을 실행하여 PyTorch, CUDA 12.8 가속, Triton Windows 및 Gemlite를 자동으로 가상환경(`.venv`)에 구축합니다.

## 7. 보안 및 주의사항
- **시크릿 키 보호:** 본 프로젝트는 로컬 루프백(`localhost`) 상에서만 통신하므로 인증 키 노출 걱정이 없습니다.
- **네트워크 비용:** 최초 setup 시 약 1.21GB 용량의 모델을 다운로드하므로 모바일 데이터가 아닌 Wi-Fi 또는 유선 인터넷 환경에서 수행하십시오.
- **발열 조심:** 이미지 생성 시 GPU 자원을 최대로 활용하므로 노트북 등 노트북 PC의 발열과 쿨러 소음이 일시적으로 커질 수 있습니다.

## 8. 테스트 방법 (임시 데이터 생성 후 삭제 방식 명시)
1. 백엔드 서버가 켜진 상태에서 `image_generator.html` 페이지를 엽니다.
2. 프롬프트에 `a yellow cute cat`과 같은 간단한 영문 프롬프트를 작성하여 테스트를 수행합니다.
3. 생성이 정상 완료되고 이미지가 브라우저상에 온전히 표현되면 검증 완료됩니다.
4. 생성 테스트용으로 저장된 임시 렌더링 파일들은 브라우저를 닫거나 다운로드 디렉토리를 정리하여 직접 정리할 수 있습니다.

## 9. 확장 가능성
- **모델 스왑:** `.env` 혹은 PowerShell 환경 변수 `$env:BONSAI_VARIANT = 'binary'` 조작을 통해 1.21GB Ternary 모델뿐만 아니라 0.93GB Binary 모델로 손쉽게 스위칭하여 성능을 실시간으로 비교 체험할 수 있습니다.
- **모바일 연동:** 동일 Wi-Fi 대역 내에서 서버 IP로 포트포워딩하여 모바일 기기(아이폰, 안드로이드) 브라우저를 통해 원격으로 접속해 이미지를 생성하게 확장할 수 있습니다.

## 10. 인스타그램 AI 오토 스튜디오 (Instagram AI Auto Studio) [NEW]

이 도구는 사용자가 입력한 주제를 기반으로 구글 Gemini AI를 활용하여 인스타그램 콘텐츠를 기획하고, 자동으로 카드뉴스 이미지와 릴스(숏폼 비디오/음성)를 생성한 뒤, 직접 본인의 인스타그램 계정에 바로 게시할 수 있게 돕는 로컬 자동화 스튜디오입니다.

### 10.1. 주요 기능
- **AI 기반 콘텐츠 자동 기획:** 주제 입력만으로 카드뉴스 본문, 해시태그, 릴스용 숏폼 대본(스크립트)이 자동 구성됩니다.
- **카드뉴스 이미지 자동 디자인 (Pillow):** 1080x1080 규격으로 제목 카드와 본문 요약 카드들을 세련되게 렌더링합니다.
- **릴스 음성 및 비디오 합성 (edge-tts, MoviePy):** 한국어 자연어 목소리(TTS)를 렌더링하고 비디오 배경과 병합하여 9:16 비율의 릴스용 비디오(.mp4)를 자동 제작합니다.
- **원클릭 인스타그램 자동 업로드 (instagrapi):** 로컬 로그인 세션을 관리하여, 피드 슬라이드 업로드 및 릴스 동영상 게시를 자동화합니다.
- **프리미엄 웹 대시보드 UI:** 생성된 텍스트와 이미지들을 웹 브라우저에서 편리하게 수정하고 원클릭으로 인스타그램에 내보낼 수 있습니다.

### 10.2. 실행 방법
1. **환경설정 (.env 작성):**
   프로젝트 루트 폴더에 `.env.template`을 복사하여 `.env` 파일을 만들고 아래 내용을 채워 넣습니다:
   - `GEMINI_API_KEY`: 구글 Gemini API 키
   - `INSTAGRAM_USERNAME`: 본인의 인스타그램 아이디
   - `INSTAGRAM_PASSWORD`: 본인의 인스타그램 비밀번호
2. **배치 파일 실행:**
   `run.bat` 파일을 더블클릭합니다. 가상환경 세팅 및 의존성 패키지가 자동 구성되고 웹 서버가 구동되며 대시보드 브라우저(`http://127.0.0.1:8000`)가 열립니다.
3. **콘텐츠 생성 및 게시:**
   - 대시보드의 주제 입력 칸에 만들고 싶은 콘텐츠 주제를 넣고 `콘텐츠 생성하기`를 클릭합니다.
   - 우측 화면에서 생성된 카드뉴스 슬라이드(피드)와 릴스 동영상을 확인합니다.
   - 글 내용을 필요에 맞게 수정한 후 `인스타그램에 즉시 게시` 버튼을 누르면 업로드가 완료됩니다.
