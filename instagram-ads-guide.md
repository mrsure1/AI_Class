## 3. Meta 광고 — 가장 쉬운 방법 (인스타 앱에서)

> 처음엔 **앱에서 "게시물 알리기"**로 시작하는 게 제일 쉽습니다.

### STEP 1. 광고할 게시물 올리기
1. 인스타 앱 → @ai.make.learn 로그인
2. 위 캡션 + A/B/C 이미지 중 하나로 **피드 게시물** 업로드
3. 게시 후 24시간 지나도 OK (바로 해도 됨)

### STEP 2. 게시물 알리기 (Boost)
1. 해당 게시물 → **알리기** (Boost Post) 버튼
2. **목표**: `프로필 방문` 또는 `웹사이트 방문`
   - 링크 클릭을 늘리려면 → **웹사이트 방문** 선택
   - URL: `https://mrsure1.github.io/AI_Class/instagram.html`
3. **오디언스**
   - **자동(추천)** — 처음 3~5일은 이걸로 테스트
   - 또는 **수동 설정** (아래 4장 참고)
4. **예산**
   - 일 예산 **1만~2만원**으로 7일 테스트 (총 7~14만원)
   - "많이 도달"보다 **링크 클릭** 또는 **프로필 방문** 지표 확인
5. **결제** → 게시

### STEP 3. 3일 후 확인
| 지표 | 좋은 신호 | 조치 |
|------|----------|------|
| 링크 클릭 | 1% 이상 | 예산 유지 or 소폭 증액 |
| 전화 문의 | 1건 이상 | 잘 되는 소재로 재광고 |
| 클릭 0.3% 미만 | | 이미지/문구 바꿔 A/B 테스트 |
