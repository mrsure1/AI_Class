② 복붙 프롬프트 30개

ChatGPT·Gemini 공통 사용 가능. 수업 순서대로 번호 매김.

③ ChatGPT · Gemini 이미지 생성

ChatGPT에서 이미지 만들기

  1. chatgpt.com 접속 → 로그인
  2. 새 대화 시작
  3. 아래 프롬프트 붙여넣기 → 전송
  4. 마음에 안 들면 "글자 더 크게" "배경 더 밝게" 등 추가 요청

※ Plus 등 유료 플랜에서 이미지 생성 가능. 수업 전 참가자 플랜 확인.

Gemini에서 이미지 만들기

  1. gemini.google.com 접속 → 로그인
  2. 대화창에 프롬프트 입력
  3. "이미지 만들어줘"가 포함되면 생성 시작
  4. 한국어로 수정 요청 가능

※ Google 계정만 있으면 사용 가능한 경우가 많음.

이미지 생성 프롬프트 (복붙용)

④ 기존 사진 교정 · 편집

사진을 첨부(업로드)한 뒤 아래 문장을 붙여넣습니다. ChatGPT·Gemini 모두 동일하게 시도.

사진 교정 진행 순서 (강사 데모용)

  1. 교정할 사진 1장 준비 (가족사진·여행사진 등)
  2. ChatGPT 또는 Gemini 대화창에서 📎 클립(첨부) 버튼 → 사진 선택
  3. 아래 프롬프트 중 하나 붙여넣기 → 전송
  4. 결과 확인 후 "조금 더 밝게" 등 한 가지씩만 추가 요청 (한꺼번에 많이 X)
  5. 완성 이미지 길게 눌러 저장 또는 다운로드

⚠️ 얼굴이 많이 바뀌거나 사실과 다른 합성은 피하도록 안내. "자연스럽게" "원본 느낌 유지"를 프롬프트에 넣으세요.

⑤ 강사 진행 팁

시니어 수업 원칙

  • 한 번에 메뉴 1~2개만
  • 10분마다 "화면 보이세요?" 확인
  • 프롬프트는 화면에 크게, 직접 복사해 따라하기
  • 실패해도 괜찮다고 격려 → 다시 질문 연습

수업 중 자주 나오는 질문

  • 틀리면? → "참고용, 전문가 확인 필요"
  • 비용? → 무료/유료 플랜 차이 간단히
  • 개인정보? → 주민번호·계좌·비번 넣지 말 것
  • 사진 이상해요 → 요청을 하나씩만 수정
