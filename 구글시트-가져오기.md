# 구글 시트에 접수 목록 올리기

파일: `AI기초교육-현수막-연락-접수.csv`

## 방법 1 (가장 빠름)

1. [Google Drive](https://drive.google.com) 접속
2. **새로 만들기** → **파일 업로드** → 위 CSV 선택
3. 업로드된 파일 더블클릭 → **Google 스프레드시트로 열기** 선택
4. 상단 메뉴 **파일** → **이름 바꾸기** → `AI기초교육 현수막 연락 접수`

## 방법 2 (빈 시트에 붙여넣기)

1. [새 스프레드시트](https://sheets.google.com) 만들기
2. A1 셀 선택 후 `AI기초교육-현수막-연락-접수.csv`를 메모장으로 열어 **1행 전체 복사** → 시트 A1에 붙여넣기
3. **데이터** → **텍스트를 열로 분할** → 구분 기호 **쉼표**

## 열 설명

| 열 | 입력 예시 |
|----|-----------|
| 접수일시 | 2026-06-04 14:30 |
| 이름 | 홍길동 |
| 연락처 | 010-1234-5678 |
| 연락경로 | 현수막 / 지인 / SNS |
| 참석가능 | 가능 / 어려움 / (공란) |
| 안내발송 | 미발송 / 접수확인 / 일정안내 / 리마인드 |
| 확정일시 | 6월 15일(일) 오후 2시 |
| 확정장소 | ○○센터 2층 |
| 비고 | 메모 |
| 설문작성일시 | 2026-06-10 14:30 (설문 제출 시 자동) |
| 컴퓨터사용능력 | 선택 항목 |
| AI경험 | 선택 항목 |
| AI하고싶은것 | 선택 항목 |
| AI기타 | 기타 직접 입력 |
| 보유기기 | 선택 항목 |
| 설문완료 | Y |

설문 연동 설정은 `구글시트-설문-연동.md` 참고.  
문자 발송 템플릿은 `문자-템플릿.md` 참고.
