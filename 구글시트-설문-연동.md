# 구글 시트 설문 연동 설정 (1회만 하면 됨)

설문 제출 시 아래 시트의 **해당 참가자 행**에만 설문 결과가 기록됩니다.  
기존 접수 정보(이름, 연락처, 참석가능 등)는 **삭제·덮어쓰지 않습니다.**

시트: https://docs.google.com/spreadsheets/d/1giowEACfpBA_3fBqyQnsXg4Tojtfw53HGDNlWGVWvrI/edit

---

## 1. Apps Script 붙이기

1. 위 구글 시트 열기
2. 메뉴 **확장 프로그램** → **Apps Script**
3. 기본 `Code.gs` 내용을 **전부 삭제**
4. 이 폴더의 `google-apps-script/Code.gs` 내용을 **복사해서 붙여넣기**
5. **저장** (Ctrl+S), 프로젝트 이름 예: `AI설문연동`

---

## 2. 웹 앱으로 배포

1. Apps Script 우측 상단 **배포** → **새 배포**
2. 유형: **웹 앱**
3. 설정
   - 설명: `AI 설문 수집`
   - 실행: **나**
   - 액세스 권한: **모든 사용자** (익명 포함)
4. **배포** 클릭 → Google 계정 권한 허용
5. 표시되는 **웹 앱 URL** 복사  
   (형식: `https://script.google.com/macros/s/xxxxx/exec`)

---

## 3. 설문 페이지에 URL 넣기

`survey.html` 안의 `SHEETS_URL` 값을 복사한 웹 앱 URL로 바꿉니다.

```javascript
const SHEETS_URL = 'https://script.google.com/macros/s/여기에_복사한_URL/exec';
```

GitHub Pages에 올렸다면 `survey.html` 수정 후 다시 push 하세요.

---

## 4. 시트에 추가되는 열 (자동 생성)

최초 설문 제출 시 기존 열 오른쪽에 아래 열이 **자동으로 추가**됩니다.

| 열 | 내용 |
|----|------|
| 설문작성일시 | 제출 시각 |
| 컴퓨터사용능력 | 선택 항목 |
| AI경험 | 선택 항목 |
| AI하고싶은것 | 선택 항목 |
| AI기타 | 기타 직접 입력 |
| 보유기기 | 선택 항목 |
| 설문완료 | Y |

---

## 5. 참가자 매칭 방식

- 설문 페이지에서 **이름 + 연락처** 입력
- 시트의 `이름`, `연락처` 열과 비교해 **같은 행**을 찾아 설문 열만 업데이트
- 연락처는 `010-1234-5678`, `01012345678` 등 형식 달라도 매칭됨
- 같은 번호로 재제출 시 설문 열만 **최신 내용으로 갱신**

### 개인별 링크로 이름·번호 미리 채우기 (선택)

문자에 아래처럼내면 참가자가 입력할 필요가 줄어듭니다.

```
https://mrsure1.github.io/AI_Class/survey.html?name=홍길동&phone=01012345678
```

## 6. 문제 해결

| 증상 | 해결 |
|------|------|
| 접수 목록에서 찾을 수 없음 | 시트에 이름·연락처가 있는지, 신청 때와 동일한지 확인 |
| 제출이 안 됨 | `SHEETS_URL`이 비어 있지 않은지, 웹 앱이 **모든 사용자**로 배포됐는지 확인 |
| 권한 오류 | Apps Script 배포 시 시트 접근 권한 다시 허용 |
| 코드 수정 후 반영 안 됨 | **배포** → **배포 관리** → 연필 아이콘 → **새 버전**으로 재배포 |

---

## 7. 테스트

1. 시트에 테스트 행 추가 (이름, 연락처)
2. `survey.html`을 브라우저에서 열고 같은 이름·연락처로 설문 제출
3. 해당 행 오른쪽 설문 열에 값이 들어갔는지 확인
