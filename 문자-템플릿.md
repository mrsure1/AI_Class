# AI 기초교육 문자(SMS) 발송 템플릿

> 사용 순서: ① 접수 확인 → (10명 모집 후) ② 일정 확정 → ③ 전날 리마인드
> `[ ]` 부분은 발송 전 실제 정보로 바꿔주세요.
> 90자(SMS) 초과 시 LMS/장문 문자로 발송됩니다.

---

## 1. 접수 확인 문자 (신청 직후 즉시 발송)

```
[AI 기초교육] 신청 감사합니다 :)
○○○님, 접수가 정상 완료되었습니다.

ChatGPT·Gemini 기초교육은
인원이 모이면(최소10명) 장소와
일정을 정해 다시 안내드립니다.

확정되는 대로 문자로 가장 먼저
연락드릴게요. 잠시만 기다려주세요!

문의: [담당자 연락처]
```

### (짧은 버전 - SMS용)
```
[AI 기초교육] ○○○님 신청 접수 완료!
인원 모집 후 장소·일정 정해지면
문자로 바로 안내드릴게요. 감사합니다.
문의 [연락처]
```

---

## 2. 일정 확정 안내 문자 (10명 모집 후 일괄 발송) ⭐핵심

```
[AI 기초교육 일정 안내]
○○○님, 기다려주셔서 감사합니다!
교육 일정이 아래와 같이 확정되었어요.

▶ 일시: [O월 O일(요일) 오후 O시~O시]
▶ 장소: [장소명 / 주소]
▶ 대상: 비전공자·시니어·학생·직장인
▶ 준비물: 스마트폰 또는 노트북
▶ 교육비: [무료 / OO원]

▶ 커리큘럼 보기: [홈페이지 주소]

참석 가능 여부를 이 번호로
[O월 O일]까지 회신 부탁드립니다.
(가능 / 어려움)

문의: [담당자 연락처]
```

## 발송 팁
- 광고성 문자(2번 일정 안내 포함)는 정보통신망법상 **`(광고)` 표기 + 무료수신거부 번호**가 필요할 수 있습니다. 단, **본인이 신청한 사람에게 보내는 안내 문자**는 일반적으로 광고 규제 예외(신청자 대상 정보성 문자)로 보지만, 안전하게 하려면 1번 접수 시 "교육 안내 문자 수신에 동의"를 받아두세요.
- 단체 발송은 문자 발송 서비스(예: 알리고, 뿌리오, 네이버 등) 또는 휴대폰 단체문자 기능 이용.
- 회신을 받기 쉽게 **"가능 / 어려움"처럼 짧게 답하도록** 유도하면 출석 집계가 편합니다.
